
  # Luxury Jewellery Recommender Page

  This is a code bundle for Luxury Jewellery Recommender Page. The original project is available at https://www.figma.com/design/wIJni0tjXmodYqDjl6orkG/Luxury-Jewellery-Recommender-Page.

## Running the code

1. Install dependencies:

```bash
npm install
```

2. Configure environment variables:

```bash
cp .env.example .env
# update VITE_API_URL if your FastAPI server uses a different host/port
```

3. Start the development server:

```bash
npm run dev
```
